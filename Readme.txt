# HematoVision - Waste Image Classification Flask App

## Project Summary:
A Flask web app using VGG16 transfer learning to classify waste images into:
- Biodegradable
- Recyclable
- Trash

## Files:
- app.py: Flask application
- healthy_vs_rotten.h5: Pretrained model
- templates/: HTML UI pages
- static/uploads/: Uploaded test images

## How to Run:
1. Install dependencies (`pip install flask tensorflow pillow`)
2. Run `python app.py`
3. Visit `http://127.0.0.1:5000/`

## Scenarios:
- Healthcare automation
- Telemedicine
- Medical education
